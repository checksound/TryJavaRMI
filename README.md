# Try Java RMI

Prima utilizzare lo script `compile.bat` per compilare i sorgenti.

## ESECUZIONE

Prima far partire l'`rmiregistry`, utilizzando lo script `startRmiRegistry.bat`, e il [TopicServer](./src/TopicServer.java) con lo script `startServer.bat`.

Per eseguire un [Subscriber](./src/Subscriber.java) si può utilizzare lo script `startSubscriber.bat`.

Per inviare un messaggio tramite il client [Publisher](./src/Publisher.java) ulilizzando lo script `runClient.bat <messaggio>`.
